# Vietnamese Language Pack for VS Code

Gói ngôn ngữ tiếng Việt cho Visual Studio Code

## Mô tả

Đây là Vietnamese Language Pack cho VS Code, dịch giao diện người dùng sang tiếng Việt. Extension này giúp người dùng Việt Nam có trải nghiệm tốt hơn khi sử dụng VS Code.

## Tính năng

- ✅ Dịch đầy đủ menu chính (File, Edit, View, Go, Run, Terminal, Help)
- ✅ Dịch các thông báo và dialog
- ✅ Hỗ trợ debug tools
- ✅ Tương thích với VS Code 1.60.0+
- ✅ Cập nhật thường xuyên

## Cài đặt

### Phương pháp 1: Từ VSIX (Khuyên dùng)
1. Tải file `vscode-language-pack-vi-2.2.0.vsix` từ thư mục releases
2. Mở VS Code
3. Nhấn `Ctrl+Shift+P` → gõ "Install from VSIX"
4. Chọn file đã tải

### Phương pháp 2: Cấu hình ngôn ngữ
1. Mở VS Code
2. Nhấn `Ctrl+Shift+P`
3. Gõ "Configure Display Language"
4. Chọn "Tiếng Việt"
5. Restart VS Code

## Sử dụng

Sau khi cài đặt:
1. Tất cả menu và giao diện sẽ hiển thị bằng tiếng Việt
2. Các thông báo lỗi và gợi ý cũng được dịch
3. Trải nghiệm coding mượt mà hơn với ngôn ngữ mẹ đẻ
